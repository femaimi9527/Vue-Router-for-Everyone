## Git删除本地分支和远程分支
### (1)删除本地分支：

#### 1.查看本地分支列表
```
git branch
```

#### 2.删除本地分支
```
git branch -d 分支名称
```

### (2)删除远程分支：

#### 1.查看远程分支列表
```
git branch -a
```
#### 2.删除远程分支
```
git push origin --delete 远程分支名称
```
